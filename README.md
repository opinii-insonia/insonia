# Bem-vindo ao InsônIA

O InsônIA é uma plataforma inteligente e intuitiva para realizar triagem e avaliação da qualidade do sono, baseada em formulários validados clinicamente (como a Escala de Epworth e o Índice de Insônia). 

Esta aplicação foi gerada em conjunto com o Dyad e possui tanto o fluxo do paciente (testes e download de relatórios em PDF) quanto o fluxo administrativo (painel para clínicas acompanharem seus Leads/Pacientes).
